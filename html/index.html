Flex-Direction

  • Define a direção dos flex-itens;
  • Padrão: Row (linha);
  • Mudamos de row para column quando estamos definindo estilos de media queries para mobile.

Tipos de Flex-Direction:

Flex-direction: row;

  • Os itens ficam em linha

Flex-direction: row-reverse;

  • Os itens ficam em linha reversa: 3,2,1

Flex-direction: column;

  • Os itens ficam em única coluna, um embaixo do outro.

Flex-direction: column-reverse;

  • Os itens ficam em única coluna, um embaixo do outro, mas em ordem reversa.

Justify-content

  • Alinha os itens flex no container ao longo do eixo principal.
  • O Justify-content não funciona em items com flex-grow definido.

Justify-content: flex-start;

  • Padrão. Os itens são alinhados a partir do início do eixo principal (container).

Justify-content: flex-end;

  • Os itens são alinhados a partir do fim do eixo principal (container).

Justify-content: center;

  • Os itens são alinhados ao centro do eixo principal.

Justify-content: space-between;

  • Cria um espaço igual entre os elementos;
  • Mantem o primeiro grudado ao início e o último ao final.

Justify-content: space-around;

  • Cria um espaço entre os elementos;
  • O espaço do meio é duas vezes maior que o inicial e o final.

Flex-Grow

  • Define a proporção de um flex-item de crescer;
  • Padrão: 0 - indica que não deve crescer;
  • Aceitos apenas valores numéricos positivos.
  • Ao definir 1 para os flex-itens, eles tentarão ter a mesma largura e vão ocupar 100% do container
  • Caso haja um elemento muito largo, ele respeitará o mesmo;
  • Ex: Tenho em uma linha, 4 itens, onde 3 são flex-grow: 1 e o último é flex-grow: 2. O que acontece?
  • R.: O flex-grow 2 tentará ocupar duas vezes mais espaço extra do que os outros elementos.
  • Obs: O Justify-content não funciona em items com flex-grow definido.

Flex-grow: número;

  • Basta definir um número.

Flex-grow: 0;

  • Obedece o width do elemento ou o flex-basis.

Flex-Basis

  • Define o tamanho inicial do item, antes da distribuição do espaço restante;
  • Dependendo da direção do eixo principal (horizontal e vertical), essa propriedade define largura ou altura mínima do elemento antes que ele seja redimensionado por outras propriedades.
  • Ao definirmos flex-grow: 1 e temos auto no basis, o valor restante para ocupar o container é distribuído ao redor do conteúdo do flex-item;

Flex-basis: auto;

  • Padrão. Faz com que a largura da base seja igual a do item;
  • Se o item não tiver tamanho especificado, o tamanho será de acordo com o conteúdo.

Flex-basis: unidade;

  • Pode ser em %, px, em, e etc.

Flex-basis: 0;

  • Se o grow for maior que 1, ele tentará manter todos os elementos com a mesma largura, independente do conteúdo (por isso 0 é o valor mais comum do flex-basis);
  • Caso contrário, o item terá a largura do seu conteúdo.

flex-basis: auto;

Flex-Shrink

  • Define a capacidade de redução de tamanho do item.

Flex-shink: 1;

  • Padrão. Permite que os itens tenham seus tamanhos (definidos a partir de width ou flex-basis) reduzidos para caber no container.

Flex-shink: 0;

  • Não permite a diminuição dos itens;
  • Um item com flex-basis: 300px nunca diminuirá menos do que 300px, mesmo que o conteúdo não ocupe todo esse espaço.

Flex-shink: número;

  • Um item com shrink: 3 diminuirá três vezes mais que um item com 1.

Flex

  • Atalho para as propriedades flex-grow, flex-shrink e flex-basis;
  • É recomendado usar a propriedade flex, ao invés dos valores separados;
  • No exemplo é possível ver as mesmas configurações do exemplo do flex-basis porém agora utilizando apenas a propriedade flex.

Flex: 1;

  • Define flex-grow: 1; flex-shrink: 1; e flex-basis: 0;
  • Em alguns browsers define como 0%, pois estes ignoram valores sem unidades, porém a função de 0 e 0% é a mesma.

Flex: 0 1 auto;

  • Esse é o padrão, se você não definir nenhum valor de flex ou para as outras propriedades separadas, o normal será flex-grow: 0, flex-shrink: 1 e flex-basis: auto.

Flex: 2;

  • Define exatamente da mesma forma que o flex: 1; porém neste caso o flex-grow será de 2, o flex-shrink continuará 1 e o flex-basis 0.

Flex: 3 2 300px;

  • flex-grow: 3, flex-shrink: 2 e flex-basis: 300px;

flex: n 1 auto;

Order

  • Modifica a ordem dos flex itens;
  • Sempre do menor para o maior;
  • Ex -> Order: 1, aparece na frente de order: 5

Order: número;

  • Número para modificar a ordem padrão;
  • Pode ser negativo.

Order: 0;

  • 0 é o valor padrão e isso significa que a ordem dos itens será a ordem apresentada no HTML.
  • Se você quiser colocar um item do meio da lista no início da mesma, sem modificar os demais, o ideal é utilizar um valor negativo para este item, já que todos os outros são 0.

Align-Self

  • Serve para definirmos o alinhamento específico de um único flex item dentro do container;
  • Esta propriedade permite sobrescrever no item o comportamento que foi definido pela propriedade align-items.
  • O alinhamento acontece tanto em linha quanto em colunas.

Align-self: auto;

  • Padrão. Respeita o comportamento definido no container por meio do align-items

Align-self: flex-start;

  • Alinha o item ao início.

Align-self: flex-end;

  • Alinha o item ao final.

Align-self: center;

  • Alinha o item ao centro.

Align-self: baseline;

  • Alinha o item a linha de base.

Align-self: stretch;

  • Estica o item.
